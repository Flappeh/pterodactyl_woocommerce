# pterodactyl_woocommerce Connector

<h3 align=center>Quick Summary</h3>
<br/>
This project is used to automatically deploy a pterodactyl server as well as manage every resource of them everytime a new purchase was done in a website using woocommerce.
<br/><br/>
REST server is used to manage server type changes inside the panel and through a proxy server running on nginx proxy manager and port the requests to respective servers securely
